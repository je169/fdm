Instrucciones:
1. Guarda la imagen que proporcionaste en el proyecto.
2. Copia/pega el archivo en esta carpeta: `public/assets/img/`.
3. Asegúrate de que el nombre sea `logo-ficoa.png` (todo en minúsculas).

Después de esto, la app usará automáticamente `/assets/img/logo-ficoa.png` que ya está referenciado en `src/app/layout.tsx`.

Si trabajas en Windows y usas Git, añade el archivo y confirma el commit:
   git add public/assets/img/logo-ficoa.png; git commit -m "Add site logo"
